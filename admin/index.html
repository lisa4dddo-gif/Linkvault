Ad Network Integration

Paste your ad codes below

In your Adsterra dashboard: Sites → Get Code. For pop-unders choose Social Bar or Direct Link. Each code block is a complete <script> tag.

In PropellerAds: My Sites → Ad Units → Get Code. For Push Notifications use the sw.min.js they provide alongside the script tag.

In HilltopAds: Websites → Ad Spots → Get Code. Copy the complete script block including any wrapper div.

Paste any HTML/script from any ad network. The code will be injected verbatim into the correct placeholders in locked.html.

3
In index.html and locked.html, make sure <script src="assets/js/site-config.js"> is loaded before script.js.
4
Upload all changed files to your server. Test a link to verify ads are showing.
5
Delete or restrict access to this /admin/ folder on your live server.
